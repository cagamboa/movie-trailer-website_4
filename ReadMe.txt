
This is the submission for Project 1 in the Full Stack Nanodegree. 

This folder contains:
	entertainment.py
	media.py
	fresh_tomatoes.html

The project assignment is:
	Create a webpage showing the favorite movies of the submitting student. 

	1. Open a terminal window
	2. Go to the directory that has the downloaded folder.
	3. Type: python entertainment.py
	4. The program will generate a webpage containing three movie images with which the user may interact.

Alternate method to run entertainment.py

	1. Open your Python editor.
	2. Open the file entertainment.py
	3. Go to the menu of your file editor and run the module. 
	4. The program will generate a webpage containing three movie images with which the user may interact.
